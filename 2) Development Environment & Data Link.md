## Some key info

- High-Speed Serial Link: The system is configured for a 115200 baud rate, minimizing data bottlenecks and ensuring that the microsecond pulses from the HC-SR04 are transmitted without delay.

- Live Monitoring: Utilizing the Serial Monitor and Teleplot, I established a real-time data link that visualizes distance fluctuations as they happen, mirroring the live graphs seen by F1 race engineers.
